# 链上数据分析入门

## 为什么链上数据分析重要？

在 Web3 交易中，链上数据是**真正的第一手信息**。K 线图会骗人，但链上数据不会。

## 必学工具

### 1. Dune Analytics（免费）

**核心功能：**
- SQL 查询链上数据
- 自定义仪表盘
- 社区分享的查询模板

**入门查询：**
```sql
-- 查询稳定币大额转账
SELECT *
FROM ethereum.transactions
WHERE value > 1000000
  AND to IN ('0xa0b8...稳定币合约')
ORDER BY block_time DESC
LIMIT 50;
```

### 2. Nansen（付费，$149/月起）

**核心功能：**
- 智能标签（Smart Money、鲸鱼标签）
- 资金流向追踪
- 代币持仓变化监控

### 3. Arkham Intelligence（基础免费）

**核心功能：**
- 实体图谱分析
- 实时警报系统
- 可视化资金流

## 识别交易信号的 5 个步骤

### 第一步：监控聪明钱

Nansen 的 "Smart Money" 标签覆盖了历史上盈利的交易地址。

### 第二步：分析大额流入

当某代币的交易所净流入突然暴增，通常是抛售信号。

### 第三步：追踪新钱包

新创建的鲸鱼钱包大规模购买往往是拉盘信号。

### 第四步：检查持仓集中度

前 10 地址持仓 > 80% → 高度控盘，风险偏高。

### 第五步：对比链上数据与价格

```
价格涨但链上活动下降 → 背离信号
价格跌但聪明钱在买入 → 抄底时机
价格横盘但链上活跃度上升 → 即将变盘
```
